## 製作狀態建議

1. 語系繁體中文，縮放 `100%`，透明度約 `94%`。
2. 深色圖使用 `charcoal`；淺色圖切到白色/淺色主題。
3. 三個 provider 全開時，Antigravity 必須是可用狀態；若沒有 live 資料，用 UI preview 或 mock 狀態。
4. 截 panel 本體即可，四周留 16-24px 空白，不要露出桌面私人資訊。
